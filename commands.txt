To create a new repository on the command line :

echo "# SUSHI WEBSITE" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Keshab311/SUSHI-WEBSITE.git 
git push -u origin main

To push an existing repository on the existing line:

git remote add origin https://github.com/Keshab311/SUSHI-WEBSITE.git 
git branch -M main
git push -u origin main
